# PR Process

Follow the workflow in `docs/pr-review-guide.md`. Every non-trivial PR produces:

| Artefact | Path |
| ----------------------- | -------------------------------------------- |
| Analysis + findings | `docs/pr-reviews/pr-{N}/round-1/review.md` |
| Findings (later rounds) | `docs/pr-reviews/pr-{N}/round-{R}/review.md` |
| Fixes per round | `docs/pr-reviews/pr-{N}/round-{R}/fixes.md` |
| Plan | `docs/plans/{date}-{topic}.md` |
| Living doc updates | updated in place (see below) |
